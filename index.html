# arcus — what is this
> a series of timed engineering trials. ode triunfal is the first.
> we run them to find exceptional builders in portugal — the ones who don't apply.
# how it works
> run ssh arcus.sh — the brief, the puzzle, the submission all live there.
> no signup, no browser. just the terminal.
# prizes
> first blood    500€  — first to solve
> best write-up  2500€ — clearest solution
# the clock
> 24 hours from the moment the trial opens. at zero, submissions close.
